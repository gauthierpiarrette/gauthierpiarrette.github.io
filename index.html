What is WinkUp?

WinkUp is a software currently in development that has for main goal to allow people with severe motor disabilities to communicate by only using their eyes. The users will be able to communicate only by blinking. More infos on the docs page coming soon!

Why using or contributing to WinkUp?

While it exists other softwares able to handle this kind of tasks, WinkUp has also for main purpose to be totally free and to be directly accessible via a web browser. WinkUp will also use state of the art machine learning methods to achieve high accuracy when interacting with users.
